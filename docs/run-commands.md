# Run Commands

- Build: `cd <path-to-ui> && bun run build`
- Test: `cd <path-to-ui> && bun run test`
- Lint: `cd <path-to-ui> && bun run lint`
- Storybook: `cd <path-to-ui> && bun run storybook`
- Playground: `cd <path-to-ui> && bun run playground`

Make sure to be in the right directory before running any of these commands.
After running the commands, make sure to terminate the process by pressing `Ctrl + C`. This will avoid getting hang up.
