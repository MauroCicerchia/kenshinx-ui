# Theming

@kenshinx/ui uses CSS variables for theming, making it easy to customize colors, border radius, and other design tokens.

## How It Works

The library ships with:

1. **CSS Variables** - Design tokens defined in `@kenshinx/ui/styles.css`
2. **Tailwind Preset** - Maps Tailwind classes to CSS variables

All components reference these CSS variables, so changing them will update the entire UI.

## Default Theme

The default theme includes light and dark modes:

```css
:root {
  /* Light mode */
  --background: 0 0% 100%;
  --foreground: 240 10% 3.9%;
  --primary: 240 5.9% 10%;
  --primary-foreground: 0 0% 98%;
  /* ... more tokens */
}

.dark {
  /* Dark mode */
  --background: 240 10% 3.9%;
  --foreground: 0 0% 98%;
  --primary: 0 0% 98%;
  --primary-foreground: 240 5.9% 10%;
  /* ... more tokens */
}
```

## Customizing Colors

Override CSS variables in your app's CSS:

```css
/* your-app/styles.css */
@import "@kenshinx/ui/styles.css";

:root {
  /* Custom blue primary */
  --primary: 220 90% 56%;
  --primary-foreground: 0 0% 100%;

  /* Custom accent */
  --accent: 200 80% 50%;
  --accent-foreground: 0 0% 100%;
}

.dark {
  /* Custom dark mode primary */
  --primary: 220 70% 60%;
  --primary-foreground: 0 0% 10%;
}
```

## Customizing Border Radius

The `--radius` variable controls the default border radius:

```css
:root {
  /* More rounded (default is 0.5rem) */
  --radius: 0.75rem;

  /* Sharp corners */
  --radius: 0;

  /* Very rounded */
  --radius: 1rem;
}
```

Components use derived values:
- `rounded-lg` = `var(--radius)`
- `rounded-md` = `calc(var(--radius) - 2px)`
- `rounded-sm` = `calc(var(--radius) - 4px)`

## Available Tokens

### Colors

| Token | Description |
|-------|-------------|
| `--background` | Page background |
| `--foreground` | Default text color |
| `--primary` | Primary brand color |
| `--primary-foreground` | Text on primary |
| `--secondary` | Secondary color |
| `--secondary-foreground` | Text on secondary |
| `--muted` | Muted backgrounds |
| `--muted-foreground` | Muted text |
| `--accent` | Accent color for highlights |
| `--accent-foreground` | Text on accent |
| `--destructive` | Error/danger color |
| `--destructive-foreground` | Text on destructive |
| `--border` | Default border color |
| `--input` | Input border color |
| `--ring` | Focus ring color |

### Other Tokens

| Token | Description |
|-------|-------------|
| `--radius` | Default border radius |
| `--chart-1` to `--chart-5` | Chart/data viz colors |

## Dark Mode

Enable dark mode by adding the `dark` class to your `<html>` element:

```tsx
// Toggle dark mode
document.documentElement.classList.toggle("dark");
```

Or use a theme provider:

```tsx
function ThemeToggle() {
  const [isDark, setIsDark] = useState(false);

  useEffect(() => {
    if (isDark) {
      document.documentElement.classList.add("dark");
    } else {
      document.documentElement.classList.remove("dark");
    }
  }, [isDark]);

  return (
    <Switch checked={isDark} onCheckedChange={setIsDark} />
  );
}
```

## Example: Brand Theme

Here's an example of a complete brand customization:

```css
:root {
  /* Brand blue palette */
  --primary: 221 83% 53%;
  --primary-foreground: 210 40% 98%;

  /* Warmer grays */
  --background: 30 20% 98%;
  --foreground: 20 14% 4%;
  --muted: 30 20% 96%;
  --muted-foreground: 20 10% 40%;

  /* Larger radius for friendlier feel */
  --radius: 0.75rem;
}

.dark {
  --primary: 217 91% 60%;
  --primary-foreground: 222 47% 11%;

  --background: 222 47% 11%;
  --foreground: 210 40% 98%;
  --muted: 217 33% 17%;
  --muted-foreground: 215 20% 65%;
}
```
